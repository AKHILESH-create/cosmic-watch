# 🛰️ AstroShield - Asteroid Tracker & Risk Analyser

A full-stack web application developed during the **24-Hour Code Relay 3.0 Hackathon** at **Pravaah'26**, organized by **Indian Institute of Technology (IIT) Bhubaneswar**.

The project aims to monitor **Near-Earth Objects (NEOs)** by fetching real-time asteroid data, analyzing potential risks, and presenting the information through an intuitive and interactive dashboard.

---

## 🚀 Problem Statement

**Interstellar Asteroid Tracker & Risk Analyser**

Develop a platform that can:
- Track Near-Earth Objects (NEOs) in real time.
- Analyze asteroid characteristics and potential threats.
- Display meaningful insights through data visualization.
- Provide an interactive dashboard for users.

---

## ✨ Features

- 🌍 Real-time Near-Earth Object tracking
- 📡 API integration for asteroid data
- 📊 Interactive dashboard
- ⚠️ Asteroid risk analysis

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/interstellar-asteroid-tracker.git
```

Move into the project folder

```bash
cd interstellar-asteroid-tracker
```

### Install frontend dependencies

```bash
cd frontend
npm install
npm start
```

### Install backend dependencies

```bash
cd ../backend
npm install
npm run dev
```
